Every constant below is exported from `@playbykey/theory`. Select inputs to see live values and the exact function call.

## Notes

`NOTES` is the ordered list of all 12 chromatic notes. `ENHARMONIC_LABELS` maps each accidental note to its flat/sharp display string.

<NotesExplorer client:load />

## Modes

`MODES` describes each of the 7 diatonic modes — name, scale degree, and character. `MODE_INTERVALS` and `MODE_SEMITONE_OFFSETS` give the step sizes and absolute offsets from the root.

<ModeExplorer client:load />

## Intervals

`INTERVAL_DEFINITIONS` maps each interval ID to a label and semitone count. `resolveIntervalEndpoints` uses this data to find the actual notes that bound an interval in a given key and mode.

<IntervalExplorer client:load />

## Scales

`SCALE_DEFINITIONS` maps each scale kind to a label and how its notes are computed. `getDerivedScaleNotes` uses this to return the correct note set for any root and scale kind.

<ScalesExplorer client:load />

## Notation and display

`NOTATION_IDS` controls how `buildNoteMap` labels each note — by name or by scale degree. Switch notation types to see how the same root and mode produce different labels.

<NotationExplorer client:load />
