## Function Reference

**Scale and mode:** `getScaleNotes`, `getParentScaleModes`, `getModeAlterations`, and `getModalRoot` derive notes and relationships from a root and mode.

**Key relationships:** `getRelativeMinorKey`, `getRelativeMajorKey`, `getKeySignatureCount`, and `getCircleOfFifthsOrder` answer common key-signature questions.

**Note utilities:** `getScaleDegree`, `isNoteInScale`, `getNoteIndex`, `noteAtIndex`, `getSemitoneDistance`, and `buildNoteMap` work with individual notes and chromatic indices.

**Type guards:** `isNote` and `isModeName` validate string inputs at runtime.
