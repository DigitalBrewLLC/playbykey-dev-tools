An `IntervalId` names one of the 13 standard intervals in the catalog, from half steps through the octave. Use `resolveIntervalEndpoints` to find the exact from/to notes for any interval in a given root and mode, or look up semitone distances directly from `INTERVAL_DEFINITIONS`.

## Interval Types

**`IntervalContext`** — `{ root: Note; mode: ModeName; interval: IntervalId }`. Passed to `resolveIntervalEndpoints` to anchor interval resolution in a scale.

**`ResolvedInterval`** — `{ from: Note; to: Note; semitones: number; label: string }`. The resolved endpoints plus the catalog label for the interval.

**`IntervalDefinition`** — `{ label: string; intervalSpec: IntervalSpec }`. Describes how an interval is defined relative to scale degrees. See the [Constants](/theory/constants/) page for the full `INTERVAL_DEFINITIONS` catalog.
