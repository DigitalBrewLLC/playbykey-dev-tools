An `IntervalId` names one of the 13 standard intervals in the catalog, from half steps through the octave. The intervals module lets you look up semitone distances and resolve concrete from/to notes for any root and mode.

## Types Reference

**`IntervalContext`** — `{ root: Note; mode: ModeName; interval: IntervalId }`. Passed to `resolveIntervalEndpoints` to anchor interval resolution in a scale.

**`ResolvedInterval`** — `{ from: Note; to: Note; semitones: number; label: string }`. The resolved endpoints plus the catalog label for the interval.

**`IntervalDefinition`** — `{ label: string; intervalSpec: IntervalSpec }`. Describes how an interval is defined relative to scale degrees. See the Constants page for the full catalog.
