`noteToMidi(note, octave)` returns the MIDI note number for a note at a given octave, using scientific pitch notation (C4 = middle C = MIDI 60). `midiToNote(midiNumber)` is its inverse, returning the note and octave for a MIDI number.

`noteToFrequency(note, octave)` returns the frequency in Hz for a note at a given octave, using equal temperament tuning (A4 = 440Hz).
